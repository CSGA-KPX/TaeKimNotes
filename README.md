# "は"和"が"的区别
src: https://weibo.com/ttarticle/p/show?id=2309404293709100384188

**“は”强调后面的部分，相反“が”强调前面。**

私の名前が秋山燿平です

私の名前は秋山燿平です

（我的名字是秋山燿平）

因为这句话是介绍名字的，所以想强调的应该是后面的“秋山燿平”。

如果用“が”的话，就是强调“私の名前は”。

不对，这句话的重点是“我的名字叫什么”，所以听起来奇怪了。因此应该用“は”
